This metaphactory instance is connected to an Amazon Neptune instance, which is preloaded with the Open PermID dataset and some interlinked extensions. The platform is pre-configured, including templates and examples to interact with this particular dataset.

License & Attribution

You are using this instance of the metaphactory platform as a trial version.

The Open PermID dataset (bulk download from 2017/11/06) is licensed from Thomson Reuters under the Creative Commons with Attribution license, version 4.0 (CC-BY). The relations and attributes under the CC-NC 4.0 license are excluded from this dataset. In addition, person entities have been enriched with an atomic rdfs:label. The labels have been constructed from the original vcard:given-name, vcard:family-name and vcard:honorific-prefix attributes as provided by the Open PermID dataset.

Data in the GeoNames sub-graph has been extracted from geonames.org and is licensed under a Creative Commons Attribution 3.0 License.

PermID organizations have been cross-linked with wikidata.org entities using the tr-org:hasLEI attribute for entity matching. Data in the Wikidata sub-graph is licensed under CC0 1.0 Universal License.
